Dos sumas

Dado un arreglo de enteros nums y un entero target, regresa los índices de los dos números que al sumarlos den como resultado target.

Cada entrada (input) tendrá exactamente una solución y no puede usar el mismo elemento dos veces.


Puedes devolver la respuesta en cualquier orden.

Ejemplos:

Input = twoSums([2,7,11,15], 9)
Output = [0,1]

Input = twoSums([3,2,4], 6)
Output = [1,2]

Input = twoSums([3,3], 6)
Output = [0,1]
