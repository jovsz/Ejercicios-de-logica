¿Divisible por el digito de la izquierda?

Crea una función que tome un numero n y revisa si cada digito es divisible por el digito que tiene a su izquierda. Por último regresa un arreglo booleano dependiendo si la condición se cumple.

Ejemplo:

Input = 73312
Output = [false, false, true, false, true]
  1. No hay digito a la izquierda de 7 = false
  2. 3/7 = false
  3. 3/3 = true
  4. 1/3 = false
  5. 2/1 = true

Nota: El arreglo siempre debe de comenzar con un valor falso
